# BioCPP
### Bioinformatics Tools Written in C++

## Matrix Metrics
[matrix-metrics.cpp](matrix-metrics.cpp)

Load a matrix of aligned genomes in order to:

- generate pairwise distance matrix
- generate sub-matrices
